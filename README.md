# Parshva Panchal's Portfolio

A professional portfolio website showcasing my skills, projects, and experience as a software developer.

## Features

- Responsive design for all device sizes
- Modern UI with smooth animations
- Sections: Home, About, Projects, Skills, Contact
- Contact form with basic validation
- Easy to customize and extend

## File Structure

```
portfolio/
├── index.html          # Main HTML file
├── style.css           # Stylesheet
├── script.js           # JavaScript functionality
├── README.md           # This file
└── assets/
    ├── profile.jpg     # Profile picture (to be added)
    └── resume.pdf      # Resume PDF (to be added)
```

## Customization

1. **Personal Information**: Update the text in `index.html` with your own details.
2. **Styling**: Modify `style.css` to change colors, fonts, or layout.
3. **Projects**: Add or modify project cards in the Projects section of `index.html`.
4. **Skills**: Update the skills lists in the Skills section.
5. **Contact**: Update contact information in the Contact section.
6. **Assets**: Add your profile picture (`assets/profile.jpg`) and resume (`assets/resume.pdf`).

## Color Scheme

The website uses a blue color scheme by default. To change the primary color, modify the `--primary-color` variable in `style.css`:

```css
:root {
    --primary-color: #2563eb; /* Change this to your preferred color */
    --secondary-color: #1d4ed8;
    --accent-color: #3b82f6;
}
```

## Development

To run the website locally:

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Make changes as needed and refresh to see updates

## Deployment

This portfolio can be deployed to any static web hosting service such as:
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting
- AWS S3 + CloudFront

Simply upload the entire `portfolio` directory to your hosting provider.
